EasyCoach: Machine Learning Engineer

Built the AI features that power a soccer coaching platform used by 400+ clubs worldwide.

Overview

EasyCoach is a soccer coaching platform used by professional and amateur clubs (including Villarreal CF) to plan training sessions, manage drills, and develop players. I joined as a Machine Learning Engineer in May 2025 and led the design and rollout of the platform's AI features, the supporting data infrastructure, and the production deploys that kept them running.

What I Built

  1. AI Drill Search
    Designed and shipped a hybrid retrieval system that lets coaches search the drill library in natural language (for example, "possession drill for U12s, half-pitch, focus on transition"). It combines a small language-model step that interprets the query with six parallel ranking signals (vector similarity, lexical match, taxonomy match, and others), merged via Reciprocal Rank Fusion. Serves search across 418 club databases.
  2. PostgreSQL Vector Database
    Designed and stood up the company's vector database from scratch, using PostgreSQL with the pgvector extension and HNSW indexing. It backs every AI feature on the platform, including the embedding pipeline that keeps drill content up to date and the per-tenant isolation that keeps each of the 418 clubs' data separate.
  3. AI Autofill
    Coaches can describe a drill in a single sentence and have the platform generate the full thing: description, coaching points, equipment list, and age-appropriate adaptations. Built on GPT-4o with a custom post-generation pass that reconciles equipment mentions across text fields. Covered by a 62-test regression suite.
  4. AI Diagram
    Turns a natural-language drill description into a tactical diagram automatically. Three-stage pipeline: a language model extracts entities and movements, a custom layout engine (with 10 formation generators) positions players and equipment, and a canvas renderer produces the final image.
  5. Drill Categorization at Scale
    Built and deployed a machine-learning drill-categorization system using a custom soccer taxonomy. Now used by 150+ teams. Ran an enrichment pipeline that re-embedded 750+ drills with taxonomy-enriched text, materially improving search relevance.
  6. YouTube Drill Ingestion
    Built a pipeline that ingests soccer drills from YouTube, enriches them with metadata, auto-categorizes them with the taxonomy model, and shares them across tenants with deduplication. This expanded the searchable library well beyond what any single club could produce on its own.
  7. Cross-Stack Data Sync
    Implemented a dual-write sync layer between the legacy PHP backend and a new Node.js service, keeping the source-of-truth MariaDB and the PostgreSQL vector database consistent across all tenants whenever a drill was created or edited.
  8. Production Ops
    Owned the production deploys and incident response for the AI coaching service running on AWS ECS.

Stack

Node.js / Express, React, PHP (CodeIgniter), MariaDB, PostgreSQL with pgvector, AWS ECS, OpenAI and Gemini APIs.
